# Installation
> [!NOTE]
> Use `git clone` instead of downloading the **ZIP** file from GitHub.
> ```
> git clone https://github.com/fr0st-iwnl/wallz.git
> ```


> [!TIP]
> To update the wallpapers, simply navigate to the folder and type:
>
> ```bash
> git pull
> ```
